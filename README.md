Este projeto consiste em uma calculadora desenvolvida em Java utilizando a biblioteca Swing para a interface gráfica. Foi o meu primeiro projeto prático realizado de forma independente, iniciado logo após meu ingresso no curso superior de Análise e Desenvolvimento de Sistemas.

O objetivo principal do projeto foi aplicar na prática conceitos fundamentais da programação orientada a objetos, lógica de programação, manipulação de eventos e construção de interfaces gráficas em Java. A calculadora realiza operações básicas como soma, subtração, multiplicação, divisão, porcentagem, inversão de sinal e raiz quadrada, com uma interface inspirada em calculadoras modernas.

Este projeto representa o início da minha jornada no desenvolvimento de software e demonstra minha evolução, dedicação aos estudos e interesse em aprender na prática desde os primeiros passos na área.
